# マークダウンプレビュー

## 概要
マークダウンをスライド形式で表示できるアプリ  


## 使い方

`./markdown.md` へスライド表示させたい内容を記述  
記述形式は通常のマークダウン書式  
ハイフン2個でページ分割  

&macr;&macr;

## 注意
基本的にはサーバへアップしないと動かない
